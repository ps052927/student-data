<h1>Student Record Manager</h1>

<p>
A simple Python project to manage student records.<br>
This application allows users to add students, calculate grades, view records, and search students by name.
</p>

<h1>Features</h1>

<p>
- Add Student Records<br>
- Calculate Percentage Automatically<br>
- Generate Grades<br>
- View All Students<br>
- Search Student by Name<br>
- Menu Driven Program
</p>

<h1>Grade System</h1>

<p>
90+ = A+<br>
80 - 89 = A<br>
70 - 79 = B+<br>
60 - 69 = B<br>
50 - 59 = C<br>
40 - 49 = D<br>
Below 40 = F
</p>
